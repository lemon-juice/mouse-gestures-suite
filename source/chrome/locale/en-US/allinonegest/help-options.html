General Preferences

Enable - enable/disable main features:
  • Mouse gestures - performed by moving the mouse with a button depressed allow you to quickly navigate through pages, open/close tabs and windows and perform many other commands without having to use toolbar buttons, menus or keyboard shortcuts. By default they are used by keeping the right mouse button down and drawing gesture shape in one of four directions: up, right, down or left. After releasing the mouse button the gesture function is executed. If you started drawing a gesture and want to cancel it just stop moving the mouse and wait at least one second before releasing the mouse button.
  • Rocker gestures - invoked by pressing one mouse button and then the other while keeping the first one pressed. By default used to go back and forward in browsing history but can be assigned to any gesture function. To customize these gestures, go to the gesture customization tab, select the gesture (at the end of the table) and click the Edit function button. Then clicking on any row will assign the function of the clicked row to the gesture.

    Functions which have a repetitive nature can be repeated without releasing the trigger button. The Allow repetition for actions that load pages advanced preference controls the repeatability of functions that load pages.
  • Scroll wheel in tab bar - you can switch between tabs by scrolling mouse wheel over the tab bar.
  • Scroll wheel navigation - perform your favorite actions by pressing a mouse button and scrolling mouse wheel at the same time.
  • Middle button scrolling - scroll pages or textareas by pressing the middle mouse button once and then moving the mouse up or down. Grab and Drag options allow you to scroll by moving the mouse while the middle button is depressed.
  • Link tooltip - see details about links as you hover over them with your mouse. Useful especially for browsing without the status bar like in full screen mode.
Mouse Gestures: Prefs - customize trail color for mouse gestures.
Link Tooltip: Prefs - options for customizing link tooltips, available only when link tooltips are enabled.
Scroll Wheel Navigation - options available when scroll wheel navigation is enabled. To use scroll wheel navigation scroll mouse wheel while keeping a mouse button pressed (the mouse button is defined in Advanced Prefs #1). Available options:
  • Perform Gesture Actions - perform actions defined in the Gestures Customization tab at the bottom of the list: Forward & Backward Scrollwheel Gesture. Any action from the list can be assigned by using the Edit function button.
  • Navigate through History - go forward/back.
  • Navigate through Tabs - switch to next/previous tab.
  • History / Tabs based on initial rotation - either Navigate through History or Navigate through Tabs depending on the initial scroll wheel rotation (clockwise or anti-clockwise).
Middle Button Scrolling - options available when middle button scrolling is enabled:
  • Mouse Gestures autoscroll - to scroll the page press and release the middle button and then move the mouse up or down. Stop autoscroll by clicking middle or left button.
  • Native autoscroll - same as above except it uses the built-in mechanism in Seamonkey instead of the one in the extension. This usually results in faster and smoother scrolling. Choose the one that works better for you.
  • MG autoscroll + Grab and Drag - both Mouse Gestures autoscroll and Grab and Drag are enabled. To use Grab and Drag keep the middle button depressed and move the mouse up or down.
  • Grab and Drag scrolling - only Grab and Drag is enabled.

Advanced Prefs #1

Trigger button - choose which mouse button will be used for mouse gestures and scroll wheel navigation.

Mouse Gesture Actions - define behavior of some gesture actions:

  • 'Open Links in Windows': Single new window with tabs - {g.linksInWindows} will open one new window with each link in separate tab.
  • 'Open new Tab/Window' opens link under performed gesture - determines whether these functions will open links in new windows/tabs.
  • 'Go Up to parent ...' / 'Load Home Page': Open in new tab - open new tab when performing these functions.
  • Rocker & Scrollwheel gestures: Allow repetition for actions that load pages - functions which have a repetitive nature (like switching tabs, going back/forward) can be repeated without releasing the trigger button.

Advanced Prefs #2

Middle Button Scrolling - define options for middle button scrolling. Availability of most of these options depend on which type of middle button scrolling is enabled in the General Preferences.

Tab scrolling - enabled to reverse scrolling direction of tabs if Scroll wheel in tab bar is enabled.

Saving and porting Mouse Gesture preferences (for advanced users)

At the moment this extension doesn't offer exporting and importing preferences but you can do it manually:

  1. Locate your profile folder.
  2. Close Seamonkey.
  3. Open prefs.js in a notepad (backup prefs.js before making any changes in case something goes wrong).
  4. Preferences for mouse gestures are those starting with "allinonegest.". Find a block of lines like this:
    user_pref("allinonegest.TTHover", true);
    user_pref("allinonegest.autoscrollCursor", true);
    ...
  5. Select and copy all lines containing mouse gesture preferences. You can save them in a new file for future use or import to another profile or Seamonkey installation.
  6. To import preferences to another profile, open prefs.js of that profile and paste the preferences into the file. If "allinonegest." preferences already exist delete them first so you don't end up with duplicates. It doesn't matter if mouse gestures are already installed or not in the new profile, just make sure Seamonkey is not running when you modify the file.

Restoring defaults

In order to restore default mouse gesture options close Seamonkey and delete all "allinonegest." preferences in prefs.js (see previous section for details). When you next run Seamonkey the default preferences will be restored.
